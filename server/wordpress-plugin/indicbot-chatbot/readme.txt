One-click install of the IndicBot AI chatbot widget on your WordPress site.

== Description ==

Add the IndicBot AI chatbot to your WordPress site without editing theme files.

1. Install and activate this plugin.
2. Go to Settings → IndicBot Chatbot.
3. Enter your **Chatbot script URL** (from your IndicBot dashboard, e.g. https://api.yourdomain.com/chatbot.js).
4. Enter your **User ID** (from the dashboard Embed section).
5. Save. The widget will load on your site.

== Installation ==

1. Upload the `indicbot-chatbot` folder to `/wp-content/plugins/`.
2. Activate the plugin in the WordPress admin.
3. Go to Settings → IndicBot Chatbot and enter your script URL and User ID.

== Frequently Asked Questions ==

= Where do I get the script URL and User ID? =

Log in to your IndicBot dashboard, open the Dashboard or Embed section, and copy the script URL and your User ID from the embed code.

= The widget doesn't appear. =

Clear your site and browser cache. Ensure the script URL is correct and your domain is allowed in IndicBot Domain Security settings.
